二叉搜索树(Binary Search Tree)

基本概念

二叉搜索树是二叉树的一种，是应用非常广泛的一种二叉树，也被称为二叉查找树或者二叉排序树

特点

任意一个节点的值都大于其左子树的所有节点的值

任意一个节点的值都小于其右子树的所有节点的值

它的左右子树也是一棵二叉搜索树

注意点

二叉搜索树存储的元素必须具备可比较性

  • 比如int,double等
  • 如果是自定义类型，需要指定比较方式
  • 不允许为空

删除节点

删除节点,我们要考虑以下三种情况,然后进行不同的操作:

  • 删除叶子节点
  • 删除度为1的节点
  • 删除度为2的节点

下面就对以上三种情况进行思路分析,然后再以代码实现

删除叶子节点

直接删除

  1. node == node.parent.left

    node.parent.left = null

  2. node == node.parent.right

    node.parent.right = null

  3. node.parent == null

    root = null

删除度为1的节点

用child替代node的位置

child 是 node.left 或者是 node.right

  1. 如果 node是左子节点

    child.parent = node.parent

    node.parent.left = child

  2. 如果node是右子节点

    child.parent = node.parent

    node.parent.right = child

  3. 如果node是根节点

    root = child

    child.parent = null

删除度为2的节点

举例: 先删除5, 再删除4

  1. 先用前驱或者后继节点的值覆盖原节点的值
  2. 然后删除相应的前驱或者后继节点
  3. 如果一个节点的度为2, 那么它的前驱和后继节点的度只可能是1和0
